# BadgeFed FOSDEM 2026 Presentation

FOSDEM presentation about decentralised badges with BadgeFed - an ActivityPub-based credentialing system for nonprofits.

## Quick Start

This presentation uses Marp for slide generation. See the [Marp documentation](https://marpit.marp.app/) for details.

```bash
npm install
npm run serve
```
